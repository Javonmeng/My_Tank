操作：鼠标左键控制发射子弹，鼠标右键控制移动；空格暂停游戏。
说明：1.子弹只会对对方造成伤害，即敌方子弹不会对敌方造成伤害，同样己方子弹也是。
          2.屏幕上最多存在己方5发子弹，敌方为50。敌方靠近己方时会发射子弹。
          3.所有坦克相碰撞均会死亡。
          4.子弹相撞会消失，尽力去躲开朝向你的子弹或者击中它。
          5.获得分数达到相应关卡要求会进入下一关卡，游戏难度会增加，共三关。
          6.三种buff：五角星为清空屏幕敌方子弹，地雷为清空屏幕敌方坦克，坦克为续命...
          7.屏幕上最多同时存在三个敌方坦克。
